The png version is probably the safest to use, because some print shops cannot handle svg transparency and add a black background instead.

## Can You Recommend a Print Shop?
- I have tried out [https://printplanet.de/](https://printplanet.de/) and they fuck up the transparency if you use the svg. If you use the png then they get it correct.

- Apart from that I have tried out [https://www.shirtinator.de](https://www.shirtinator.de), but only for a different logo and a white T-shirt.

Let me know if you find a good print shop with good quality T-shirts and good handling of transparent backgrounds.
